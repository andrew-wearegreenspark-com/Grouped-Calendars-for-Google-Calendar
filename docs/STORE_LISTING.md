# Chrome Web Store listing

## Name

Grouped Calendars for Google Calendar

## Summary

Organise Google calendars into coloured groups and switch between focused, reversible calendar views.

## Detailed description

Grouped Calendars adds an organised calendar section directly to the Google Calendar sidebar.

Create coloured groups, assign calendars from My calendars or Other calendars, and control visibility without changing or deleting the original Google calendars.

Use Solo to focus on one complete group. Use Quick Solo when you want a temporary selection drawn from several groups: choose calendars first, apply the set once, then Restore the exact visibility state you had before.

Features include:

- Coloured, collapsible calendar groups
- Individual and group visibility controls
- Reversible Solo and Quick Solo modes
- Exact visibility restoration
- Searchable group-management page
- Configuration import and export
- Multiple Google account isolation
- Large-calendar-list storage fallback
- Light and dark mode support

Privacy is deliberately simple: there is no analytics, advertising, tracking, or developer server. Event content is not read or stored. Configuration remains in Chrome storage.

This extension is independently developed and is not affiliated with or endorsed by Google.

## Single-purpose statement

The extension organises Google Calendar's native calendar list into user-defined groups and provides reversible visibility controls for those calendars.

## Permission justification

### storage

Required to save group names, colours, ordering, calendar assignments, behaviour settings, and temporary Solo restoration snapshots.

### calendar.google.com host access

Required to display the Grouped section inside Google Calendar and operate the user's existing native calendar visibility controls. The extension does not request access to other websites.
